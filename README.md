# bihar_election_votes_2025_Analysis

This project contains a sample dataset and Python notebook for analyzing Bihar election results.

## Includes
- Dataset (`bihar_election_sample.csv`)
- Jupyter Notebook (`bihar_election_sample.ipynb`)
- Visualizations (line, bar, pie, histogram)

## Highlights
- Party vote trends by year
- Vote share distribution
- Turnout analysis

## Tools
Python, Pandas, Matplotlib, Jupyter Notebook
